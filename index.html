Wallapet

Aplicación Android para la compra/venta y adopción de animales.

View project on GitHub

What is wallapet?

WallaPet is an Android application to deal between buyers and sellers of animals and people who wants to put in adoption his pet.

WallaPet is going to be available at Play Store in a few weeks.

Official site and Wiki

You can check our official site here. Also you can view our repository pages in Github pages.

The project wiki has been moved from github to our server (externally) using "MediaWiki". The wiki is available in: wiki.wallapet.com. Check it out!

Technologies

The following technologies are being used in the development of Wallapet:

- JSON

- GSON

- MySQL Community Edition

- PHP

- Apache Tomcat

- HTML 5

- JavaScript

Dependencies

Wallapet app needs the following libraries to work succesfully:

- com.android.support:appcompat-v7:22.0.0

- com.google.code.gson:gson:2.2.4
